
已初始化后再次提交命令：
# 1. 查看变更（养成习惯，避免误传）
git status

# 2. 添加所有改动的文件
git add .

# 3. 提交并写上简短的改动说明
git commit -m "描述你这次改了什么，例如：修复登录bug、添加搜索功能"

# 4. 推送到 GitHub
git push origin main
